Epochs folderÂ¶

An experiment can be separated into one or many logical intervals, such as sub-experiments or trials. Those intervals are called epochs and are represented in the epochs folder. Each epoch can have zero or more TimeSeries associated with it, for example acquisition or stimulus. Associating a TimeSeries with an epoch is optional, and the number (and name) of these associated TimeSeries is user-defined. The epoch defines a window in each of these TimeSeries indicating where the TimeSeries overlaps with the epoch.
