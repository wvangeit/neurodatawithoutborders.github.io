General folderÂ¶

This is a metadata folder storing information that is important for interpreting the experiment, such as animal strain, experimental protocol and devices used. The data is stored as free-form text fields and many can be populated with excerpts from the Methods section of a paper. Metadata about the data itself (core metadata) is stored with the data itself, whether in TimeSeries or Modules.

Machine-readable fields for general metadata (ie, for interpreting the experiment) are to be included as attributes of the more general text-based fields defined below. For example, ‘related_publications’ will have an attribute (text array) for ‘pmid’ to list all Pub-Med IDs that are associated with the data in the file. The field ‘experimenter’ will have a text array ‘name’ for the names of all experimenters, and an ‘ORCID’ field (equal length text array) as well. The list, definition and controlled vocabulary for these machine-readable tags must be further developed and reviewed.

Not all fields are relevant to all experiments. Non-relevant fields can be blank or omitted.
